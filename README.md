# Blinkit Analytics Dashboard
### Project Overview
The **Blinkit Analytics Dashboard** is designed to track and analyze the sales performance of a quick-commerce platform. By visualizing transactional data, the dashboard provides insights into outlet efficiency, item popularity, and customer purchasing trends.

### Purpose
The goal of this project is to analyze:
* **Sales Performance:** Monitoring total revenue, average sales, and order volume.
* **Inventory Distribution:** Identifying top-selling item categories and fat content preferences.
* **Outlet Efficiency:** Comparing performance across different outlet types and location tiers.
* **Temporal Trends:** Tracking sales fluctuations across quarters, months, and days.
